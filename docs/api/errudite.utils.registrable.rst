errudite.utils.registrable
==========================

``Registrable`` is inspired by 
`allennlp.common.registrable <https://allenai.github.io/allennlp-docs/api/allennlp.common.registrable.html>`_.
It is a "mixin" classes for endowing base class with a named 
registry for its subclasses and a decorator for registering them.
